# A Starter Project using Laravel 5 & Vue.js

## Installation

```bash
git clone https://github.com/sohelamin/laravel-starter.git
cd laravel-starter
composer install
npm install
php -r "copy('.env.example', '.env');"
php artisan key:generate
php artisan migrate
php artisan db:seed

npm run dev
php artisan serve
```
